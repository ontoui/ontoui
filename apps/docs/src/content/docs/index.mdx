## Built for agents

OntoUI is designed with LLM agents in mind. Components are tightly-scoped and composable, with predictable props and minimal surface area — making them straightforward for agents to discover, use, and compose without ambiguity.

CSS Modules keep styles encapsulated with no global style pollution, and zero runtime dependencies beyond React.
